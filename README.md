# FTP Exposure Lab (Isolated & Educational)

This repository demonstrates a **step-by-step security assessment workflow**
to understand the risk of an exposed FTP service in an **isolated lab environment**.

The focus is on:
- Network reconnaissance
- FTP service discovery and enumeration
- Evidence collection
- Defensive security takeaways

## Scope
- Educational use only
- Authorized testing in an isolated lab
- No exploitation or credential attacks

## Audience
- Cybersecurity students
- SOC analysts
- Blue team practitioners
- Security researchers

## Disclaimer
This project must be used **only on systems you own or have explicit permission to test**.
Unauthorized testing is illegal.
